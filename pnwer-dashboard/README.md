# PNWER Dashboard Frontend

This directory contains the Vite + React frontend for the PNWER Tariff Dashboard.

For full project setup, local development, and deployment instructions, see the root README:

- [README.md](/D:/Tariff/README.md)

## Quick Commands

```bash
npm install
npm run dev
npm run build
```

## Deployment

- Frontend hosting: Vercel
- Required env var: `VITE_API_BASE`
